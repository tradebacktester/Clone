## Architecture

- **Library**: `lib/market-analysis/src/learning/feature-importance/` — 8 files (types, calculator, interaction-analyzer, confidence-learning, ranking-engine, validator, history-store, report-generator)
- **DB**: 4 tables — `fi_analysis_cycles`, `fi_feature_records`, `fi_interaction_records`, `fi_confidence_history`
- **Routes**: `artifacts/api-server/src/routes/feature-importance.ts` — 7 endpoints
- **Dashboard**: `artifacts/dashboard/src/pages/feature-intelligence.tsx` — 5-tab view
- **Nav**: "Feature Intel." in AI Engine group, `/feature-intelligence`
- **Report**: `FEATURE_IMPORTANCE_REPORT.md` in project root (generated by GET /api/learning/features/report)

## Key Design Decisions

**Why advisory only:** Engine reads from `learningFeaturesTable` (same as learning engine / pattern performance). No writes to strategy config, no auto-tuning.

**Why ConfidenceTier excluded from selective export:** `learning-confidence/confidence-engine.js` already exports `ConfidenceTier`. The feature-importance types define its own internal `ConfidenceTier` with identical values — do NOT re-export it from `lib/market-analysis/src/index.ts` to avoid collision.

**Why MIN_SAMPLE_SIZE conflicts:** Feature importance defines `MIN_SAMPLE_SIZE` (5); pattern performance defines `MIN_EVIDENCE_SAMPLE`. Export them with different names when needed.

**Route ordering:** `/learning/features/status` and `/learning/features/report` MUST be registered BEFORE `/learning/features/:id` in the Express router to prevent parameterized route swallowing.

## Interactions

10 pre-defined synergy combos (see `interaction-analyzer.ts`). Synergy = interaction win rate / individual feature baseline; >1.1x = synergistic. Uses `SYNERGY_THRESHOLD = 1.1`.

## Data Flow

1. Run learning cycle → populates `learningFeaturesTable`
2. `POST /api/learning/features/analyze` → reads from `learningFeaturesTable`, computes 17 features + 10 interactions, stores to `fi_*` tables + in-memory store
3. Dashboard reads from in-memory store via GET endpoints
